# Components Documentation

This directory contains documentation for all reusable components in the Ansopedia Creator Studio.

## Core Components

- Navigation
- Forms
- Layout
- UI Elements

## Component Guidelines

All components should follow our Component Guidelines for consistency and maintainability.

## Directory Structure

Components are organized into the following categories:

- ui/: Basic UI components (buttons, inputs, etc.)
- layout/: Layout components (sidebar, header, etc.)
- forms/: Form-related components
- global/: Globally used components
