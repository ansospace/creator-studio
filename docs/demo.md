# Ansopedia Demo

A demo of the Ansopedia platform, built with Next.js, TypeScript, Tailwind CSS, Radix UI, and React
Query.

## Inspiration

The inspiration for Ansopedia came from recognizing several critical challenges in the current
educational landscape:

**Access Gap:** Many students struggle to find quality learning resources and internship
opportunities that match their skills and career goals, particularly in computer science and related
fields.

**Information Overload:** Students often face difficulties managing large amounts of educational
content effectively, making it challenging to focus on the most relevant materials.

**Language Barriers:** Educational content is often limited by language, restricting access for
non-English speaking students and reducing global accessibility.

## What it does

Ansopedia is a comprehensive learning platform that offers:

### 1. Interactive Learning

- Personalized quiz system with progress tracking and immediate feedback
- Dynamic learning paths based on user performance
- Real-time assessment and explanations

### 2. Content Management

- Blog system for educational content
- AI-powered content summarization
- Efficient content organization and discovery

### 3. Global Education Access

- Multilingual support for content
- Worldwide educational institution database
- Cross-cultural learning opportunities

### 4. User Experience

- Dark/light mode support
- Responsive design for all devices
- Intuitive navigation and accessibility features

## How we built it

The platform was built using modern web technologies:

- Frontend Framework: Next.js 15.0.3 with TypeScript for type safety
- UI Components: Custom components built with Radix UI and styled with Tailwind CSS
- State Management: Redux Toolkit for global state management
- API Integration: React Query for efficient data fetching and caching
- AI Integration: Custom AI provider system for content enhancement

## Challenges we ran into

1. **AI Integration Complexity:** Implementing reliable AI features for content summarization and

2. **Cross-Platform Compatibility:** Ensuring consistent user experience across different devices

3. **Performance Optimization:** Balancing rich features with application performance and load

4. **Content Management:** Creating an efficient system for organizing and delivering educational
   content at scale.
